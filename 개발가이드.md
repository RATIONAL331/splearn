# Splearn 개발 가이드
## 아키텍처
- 헥사고날
- 도메인 모델 패턴
### 계층
- Domain
- Application
- Adapter

> Actor -> Adapter -> Application -> Domain


## 패키지
- `domain`: 도메인 모델
- `application`
  - `required`: 애플리케이션이 필요로 하는 포트
  - `provided`: 애플리케이션이 제공하는 포트
- `adapter`
  - `persistence`: 영속성
  - `integration`: 외부 시스템 (메시징, 다른 서버 등)
  - `webapi`: 웹
  - `security`: 보안
